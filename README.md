
# GO PONG

## Introduction
Classic game of pong developed in:
- golang
- ebiten library

The game uses a very basic AI for the opposite paddle

## Instructions
### Prerequisites
- Have go installed

### Run
Run in the your favorite terminal in the go-pong folder:
`go run *.go`
